## ðŸš€ Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/KurutoDenzeru/Inkoro.git
cd Inkoro
```

### 2. Install dependencies

```bash
# With npm
yarn install
# or
npm install
# or
bun install
```

### 3. Run the development server

```bash
npm run dev
# or
yarn dev
# or
bun run dev
```

Open [http://localhost:5713](http://localhost:5713) to view the app.
